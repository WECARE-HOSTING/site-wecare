# Project Specs — Site Proprietários Design

## O que é
Site institucional/marketing para proprietários, com foco em design premium e animações modernas usando GSAP.

## Quem usa
Visitantes externos — proprietários ou potenciais clientes.

## Tech Stack
- **Framework:** Next.js (App Router) — TypeScript
- **Estilização:** Tailwind CSS
- **Animações:** GSAP
- **Backend:** Supabase (quando necessário)
- **Deploy:** Vercel
